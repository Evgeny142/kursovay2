# Информационная система визуализации и анализа данных продаж интернет-магазина

## Назначение проекта
Данный проект представляет собой модуль визуализации и анализа данных продаж интернет-магазина. Приложение позволяет загрузить CSV-файл с данными о продажах, выполнить их обработку, рассчитать основные показатели и отобразить результаты в виде таблиц и диаграмм.

## Используемые технологии
- Python
- Streamlit
- Pandas
- Plotly

## Состав проекта
- `app.py` — основной файл приложения;
- `requirements.txt` — список зависимостей;
- `sales_data_example.csv` — пример входных данных;
- `README.md` — описание проекта и инструкция по запуску.

## Требования к входным данным
CSV-файл должен содержать следующие столбцы:
- `date` — дата продажи;
- `product` — наименование товара;
- `quantity` — количество проданных единиц;
- `price` — цена за единицу.

## Порядок запуска приложения
1. Скачайте файлы проекта.
2. Откройте терминал в папке проекта.
3. Создайте виртуальное окружение:
   - Windows:
     ```bash
     python -m venv venv
     venv\Scripts\activate
     ```
   - macOS / Linux:
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```
4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
5. Запустите приложение:
   ```bash
   streamlit run app.py
   ```
6. После запуска откройте ссылку, которая появится в терминале, и загрузите CSV-файл с данными о продажах.

## Основные функции приложения
- загрузка данных о продажах из CSV-файла;
- проверка корректности входных данных;
- фильтрация по товарам и периоду;
- расчёт основных показателей;
- построение диаграмм и графиков;
- формирование сводной таблицы.
